# **App Name**: VolunteerConnect

## Core Features:

- Project Listing: Allow non-profits to list IT projects with detailed descriptions, required skills, and estimated time commitment.
- Volunteer Profiles: Enable volunteers to create profiles highlighting their skills, experience, and availability.
- Project Selection: Allow volunteers to select projects they want to work on without an approval process.
- Communication Portal: Provide a secure messaging system for non-profits and volunteers to communicate about project details and progress.

## Style Guidelines:

- Primary color: Forest green (#337A2F), symbolizing growth and community, which goes well with non-profit work.
- Background color: Very light green (#F0FAF0), creating a calm and inviting atmosphere.
- Accent color: Warm orange (#D97706) to highlight key actions and calls to action.
- Font pairing: 'Inter' for body text (a sans-serif), paired with 'Alegreya' (a serif) for headlines to offer a balance of readability and elegance.
- Use simple, clear icons to represent project categories and skill sets.
- Design a clean, card-based layout for project listings and volunteer profiles for easy browsing.
- Subtle animations when favoriting or applying to projects
